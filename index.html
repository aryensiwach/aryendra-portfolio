Case Study: Cortex-V

The Challenge

To engineer a high-speed, scalable image retrieval system capable of finding visually similar images within a vast database based on their content, not just metadata.

My Solution

I architected the end-to-end pipeline, leveraging OpenAI's CLIP model to transform over 2,000 images into vector embeddings. For the core search functionality, I implemented a highly optimized FAISS index and developed a responsive Flask web app for real-time queries.

The Impact

The system achieved an average query latency of just 90ms with a 99% successful retrieval rate. The architecture is built to scale, handling over 100,000 images with minimal performance degradation.

Case Study: AI Code Evaluation

The Challenge

To build a platform for real-time code compilation and AI-assisted evaluation, aiming to enhance accuracy and significantly reduce the manual effort involved in grading code submissions.

My Solution

I engineered the core C++ module that seamlessly integrated Judge0 for compilation and the Gemini API for intelligent feedback. My focus was on optimizing API calls and architecting a scalable backend that was stress-tested with over 200 concurrent users.

The Impact

The platform successfully cut evaluation latency by 30% and improved developer productivity by 20%. Most notably, it achieved a 25% increase in evaluation accuracy compared to traditional rule-based systems.

Case Study: Face Recognition Attendance

The Challenge

To automate the time-consuming and error-prone process of manual attendance, particularly for large groups, and to eliminate the possibility of proxy attendance.

My Solution

I developed a Python-based system using OpenCV for real-time face detection and recognition. The process involved training a model on a custom dataset, integrating it with a CSV backend for attendance logging, and building a simple Tkinter UI for system management.

The Impact

The system achieved a 98% accuracy rate in controlled environments, reduced attendance marking time by over 95%, and effectively eliminated proxy attendance issues, ensuring an accurate and efficient record.
