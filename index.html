Case Study: Cortex-V

The Problem

Create a high-speed, scalable image retrieval system that can find similar images from a large database based on content, not just tags.

My Role & Process

I architected the end-to-end pipeline, using OpenAI's CLIP model to convert over 2,000 images into vector embeddings. I then implemented a highly optimized FAISS index for similarity search and developed a responsive Flask web app for real-time queries.

The Outcome

Achieved an average query latency of just 90ms with a 99% successful retrieval rate. The architecture is scalable to over 100,000 images with minimal performance degradation.

Case Study: Face Recognition Attendance

The Problem

Automate the manual attendance process to reduce errors and save time, especially for large classes or organizations.

My Role & Process

I developed a Python-based system using OpenCV for real-time face detection and recognition. I trained a model on a custom dataset, integrated it with a CSV backend for attendance logging, and built a simple UI with Tkinter for system management.

The Outcome

The system achieved a 98% accuracy rate in controlled environments. It reduced the time for attendance marking by over 95% and eliminated proxy attendance issues.
